SENDO ASSIM, FOI ESTABELECIDO QUE CADA TECNOLOGIA TINHA SUAS RESPONSABILIDADES FOI DIVIDO EM 3 PARTES,SEGUINDO OS PADRÕES:

- HTML: RESPONSÁVEL PELO CONTEÚDO DAS PÁGINAS. 
	É APENAS UM ARQUIVO DE TEXTO/ COM MARCAÇÕES HTML QUE DÃO SIGNIFICADOS AO 	CONTEÚDO.
		ex: TÍTULO vs PARÁGRÁFO
			<h1> Isso é um título</h>
			<p> Isso é um parágrafo</p>	

		ex:	<h1> Isso é um título com <b> negrito</b></h>
			<p> Isso é um parágrafo com <b> negrito</b></p>

- CSS: RESPONSÁVEL PELA PARTE DO VISUAL DAS PÁGINAS.
- JS: RESPONSÁVEL PELA INTERAÇÃO DAS PÁGINAS.

O QUE É UMA PÁGINA vs O QUE É UM SITE:
	PÁGINA NA WEB = SIMPLESMENTE UM ARQUIVO PRONTO HTML
		cada arquivo HTML vai ter suas particularidade, como JS, CSS, png, 		entre outros.

	SITE = NADA MAIS É DO QUE UMA PASTA DO QUE UM DIRETÓRIO COM VÁRIOS ARQUIVOS 		HTML LÁ DENTRO.

COMUNICAÇÃO CLIENTE SERVIDOR

	CLIENTE = SOMOS NÓS, QUANDO A GENTE ABRE O BROWSER(NAVEGADOR) E FAZ UMA 			BUSCA POR ALGO OU ABRE ALGUMA PAGINA ESPECÍFICA.
	
	SERVIDOR = É UMA MÁQUINA CONECTADA NA INTERNET 24 HORAS POR 7 DIAS POR 			SEMANA, QUE VAI SERVIR ESTÁ INFORMAÇÃO PARA A GENTE.

PARA QUE O CLIENTE TENHA ACESSO AO CONTEÚDO QUE ELE DESEJA, IRÁ PESQUISAR NO BROWER ONDE ESTÁ AQUELA INFORMAÇAO, SENDO ASSIM, ACHAR SERÁ ENVIADO A ELE POR HTML. PARA ISSO FUNCIONAR, É NECESSARIOS ALGUNS PASSOS:
	
	URL:LOCALIZADOR DE RECURSOS
	HTTP:PROTOCOLO DE TRANSFERÊNCIA DE HIPERTEXTO
	HTML:LINGUAGEM DE MARCAÇÃO DE HIPERTEXTO 
		(!!! HTML NÃO É UMA LINGUAGEM DE PROGRAMÇÃO, LOGO, VOCÊ NÃO PROGRAMA 			EM HTML, É UMA LINGUAGEM DE MARCAÇÃO.!!!)
